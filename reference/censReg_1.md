# censReg_1

Model of class censReg

## Usage

``` r
censReg_1
```

## Format

An object of class `censReg` (inherits from `maxLik`, `maxim`, `list`)
of length 20.

## Examples

``` r
if (FALSE) {
censReg(
  affairs ~ age + yearsmarried + religiousness + occupation + rating,
  data = Affairs

}
```
